# UNBANS Nuxt 3 Project Documentation

## Setup Instructions

1. **Clone the repository:**  
   ```bash
   git clone https://github.com/sbansss/your-repo-name.git
   cd your-repo-name
   ```  

2. **Install dependencies:**  
   ```bash
   npm install
   ```  

3. **Run the development server:**  
   ```bash
   npm run dev
   ```  

## Build Instructions

To create a production build of the application, run:

```bash
npm run build
```

This will generate the necessary files in the `.output` directory.
